# Reasons To Quit Smoking API

## Purpose

The purpose of this project is to serve as an additional API to make a callout to in my capstone project for Code:You's Software Development pathway, as well as an opportunity for students from the Web Development pathway to make their first GitHub contribution.

## Running Locally

Required Tools

- VS Code
- NodeJS
- npm

In GitBash:

1. Verify NodeJS and npm are installed on your machine by running `node -v` and `npm -v`. If a version number is returned for each command, these have been successfully installed.
2. `cd` into the directory of your choice and run `https://github.com/dan-collins-dev/reasons-to-quit-smoking-api.git`
3. `cd` into the cloned repo and run `npm install` to install the required packages.
4.  Open the project with `code .`
5.  Run the server with `npm run dev`

## Contributing

To contribute to this project, follow these steps

1. Fork the repository and clone your fork down.
2. Create a new branch called `feat/reasons` using `git checkout -B feat/reasons`
3. Update the array of reasons inside `reasons.js`
4. Add, commit, and push your changes on the current branch using `git push origin feat/reasons`
5. Either click the create a pull request message that appears in the terminal after you make your push to GitHub, or navigate to your fork's `feat/reasons` branch and create a pull request there.
